# Markdown Notes App

A front-end notes app where users can create, edit, search, tag, and preview markdown notes while saving everything locally.

## Live Demo

https://jfoster294.github.io/markdown-notes-app/

## Features

- Create notes
- Edit notes
- Delete notes
- Search notes
- Add tags
- Filter notes by tag
- Pin important notes
- Markdown live preview
- Save notes with local storage
- Responsive design for mobile and desktop

## Markdown Support

- Headings
- Bold text
- Italic text
- Inline code
- Blockquotes
- Basic unordered lists

## Built With

- HTML
- CSS
- JavaScript
- Local Storage

## Project Goal

This project was built as part of my front-end developer portfolio to practice CRUD functionality, markdown preview, search, tags, pinned notes, local storage, and responsive design.
